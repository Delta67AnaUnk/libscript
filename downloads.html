Downloads

There is no official release yet, but you can keep track of the ongoing development by checking out the current state of the Git repository. You can do so issuing the following command:

git clone https://github.com/hishamhm/libscript

It will download the sources for libscript and the various plugins. There is a Makefile at the top directory which builds all libraries and runs some tests.

You can also browse the Git tree online.
